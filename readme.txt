项目内容：freeRTOS，硬件i2c+dma驱动oled，多路adc采样（光敏，热敏模拟），，红外遥控和接收
单总线ds18b20 
pwm控制灯或舵机
spi的w25q64（功能待定）

后续：botloader+ota
//************************************项目概述*****************************************/
使用了freertos（多任务，队列，队列集，软件定时器，信号量，任务调度），具备独立bootloader+app分区，具备IAP串口升级（ota还在研究预计再1周可以完成），硬件i2c+dma驱动oled提供gui界面显示，dht11提供湿度，ds18b20单总线协议提供温度，两路模拟传感器adc+dma采集，usart1调试打印信息，usart2与esp01s通讯连接onenet平台上报传感器数据和接收下发命令，usart2采用中断空闲+dma+环形缓冲区处理，pwm控制led灯和舵机，at24c02用来储存关键信息，w25q64存储备份app区，输入上有红外遥控（定时器输入捕获）+多按键控制（按键采用外部中断+rtos的软件定时器实现单击和长按）。有数据采集任务，显示任务，输入控制任务，uart2接收处理任务，云平台通讯等任务。
